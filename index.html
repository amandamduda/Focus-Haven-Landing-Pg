FOCUS HAVEN

Focus Haven is the world-famous Internet, social media, and app blocker. Focus Haven works on your iPhone, iPad, and Mac and Windows computers. It is the best resource to give you back productivity and solitude.

FEATURES

Custom Checklist

Safe Haven allows you to create a checklist and set a time limit to complete it by, giving you that extra boost to get things done.

Location Based Blocker

We know that setting a lot of Havens can be cumbersome, so we have set up a location based features, that allows your phone or computer GPS to create a custom radius for you to work peacfully

Set Beacons

Safe Haven uses beacon technology so you can easily set it to automatically start whenever your near your work space, wether that be at home, the office, or somewhere else.

Pre-Set Havens

Know your going to need to do extra work at home tonight? No problem. You can pre-set your havens so that it will automatically start at the date and time you have set up.

Auto Reply Messages

Once you have entered your haven any calls, text messages, or chats will receive an auto response that you are currently unavailable. While we have a generic message, you can personalize it at any point in your settings.

Custom App Blocker

Haven allows you to pick and choose which apps and websites you want blocked and the length of time for each.

HOW IT WORKS
